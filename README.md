# xsshelp-py

一个辅助挖掘xss漏洞的工具（xsshelp的python版）

```
usage: xsshelp.py [-h] [-u U] [-t T]

some help

optional arguments:
  -h, --help  show this help message and exit
  -u U        The target url
  -t T        The thread number
```
